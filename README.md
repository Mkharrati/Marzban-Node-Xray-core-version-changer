 # Marzban-node xray core updater

* install
```
sudo sh -c "$(curl -sL https://github.com/Mkharrati/Nodex/raw/refs/heads/main/install.sh)" @ install
```

## How to use?

command structure :

Nodex [VERSION]

**e.g:**

Replace your desired version :
```
Nodex 1.8.24
```
